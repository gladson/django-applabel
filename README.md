django-applabel
===============

Rename Django app label in admin

Instalation
-----------

Instalation is very simple over pip.

    $ pip install django-applabel

Usage:
add settings.py

    INSTALLED_APPS += ("applabel", )

edit myapp/admin.py

    from applabel import rename
    rename("myapp", "My Application")
